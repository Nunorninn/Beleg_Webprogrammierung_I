# Beleg_Webprogrammierung_I

Ein interaktives Lernprogramm für Webprogrammierung I.

## GitHub Codespaces öffnen

GitHub Codespaces ermöglicht es dir, direkt im Browser in einer vollständigen Entwicklungsumgebung zu arbeiten – ohne lokale Installation.

### So öffnest du Codespaces:

1. Gehe auf die Hauptseite dieses Repositorys auf GitHub:  
   `https://github.com/Nunorninn/Beleg_Webprogrammierung_I`

2. Klicke auf den grünen **`<> Code`**-Button (oben rechts über der Dateiliste).

3. Wähle den Reiter **„Codespaces"**.

4. Klicke auf **„Create codespace on main"** (oder den aktuellen Branch-Namen), um eine neue Umgebung zu starten.

5. Der Codespace öffnet sich in VS Code im Browser. Nach dem Start kannst du den eingebauten Live-Server nutzen, um die Anwendung auszuführen:
   ```bash
   live-server
   ```
   Ein Browserfenster mit dem Lernprogramm öffnet sich automatisch.

### Voraussetzungen

- Ein GitHub-Konto  
- Zugang zu GitHub Codespaces (kostenlos verfügbar für öffentliche Repositorys)

## Lokale Entwicklung

Alternativ kannst du das Projekt lokal klonen:

```bash
git clone https://github.com/Nunorninn/Beleg_Webprogrammierung_I.git
cd Beleg_Webprogrammierung_I
```

Öffne anschließend `index.html` in einem Browser oder nutze einen lokalen Webserver wie `live-server`.
